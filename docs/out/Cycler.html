Class: Cycler

Cycler()

Compositions are written by defining Nodes that generate and/or transform Events (usually notes), and adding these Nodes first to Cycler.root, and then to children of root, and so on recursively. All compositions must contain an instance of Cycle (the only Node that actually sends Events to Csound); by default, the root Node of this is a Cycle.

new Cycler()

The constructor takes a defined instance of Csound that has already compiled its orchestra and is already performing.
